# Minimo Minimal Blog

A minimalistic blog site inspired by the provided design. This site features a clean layout, thin sans-serif typography, a simple logo/title, a horizontal navigation menu, a large featured image for the main post, and a grid/list of smaller posts below. All images and text are placeholders.

## Features
- Responsive, clean white background
- Simple logo/title at top left
- Horizontal navigation menu
- Large featured image for main post
- Grid/list of smaller posts
- Minimal CSS, no frameworks

## Usage
Open `index.html` in your browser. Replace placeholder images and text as needed.
